\ Stack

La stack (ou pile) est une méthode en Algorithmie, elle est beaucoup utilisée notamment dans le web, que nous verrons plus tard.
Concretement une stack est un empilement, les méthodes qui lui sont associées permettent d'y ajouter un élément, et de retirer le dernier élément ajouté.
Cela fonctionne un peu comme une pile d'assiettes, vous pouvez en rajouter une, et enlever la dernière.
La maniere la plus simple d'implémenter une stack est d'utiliser une méthode d'array.
La stack possede une complexité de O(n).
A savoir que la librairie standard possède une méthode de stack.


Applications :

- Web:
 Les bouttons en haut a gauche de votre navigateur permettant d'aller aux onglets précédents fonctionnent comme une stack.
 Votre navigateur va enregistrer tout vos onglets, lorsque vous cliquez sur le boutton retour, ce dernier vous affichera l'élément le plus en haut de la stack.
- Algorithmes:
 Permet par exemple d'inverser une string donnée.


Logique d'une stack :

 1. Création d'un pointer "TOP" pour garder la trace de l'élément le plus en haut.
 2. Lorsque l'on initialise la Stack,on lui donne pour valeur "-1" afin de vérifier que cette dernière est vide.
 3. Lorsque l'on ajoute un élément, on incrémente la valeur du pointeur "TOP" et placer le nouvel élément a la position pointée par "TOP".
 4. Quand on enlève un élément, on retourne la valeur pointée par "TOP" et réduisons la valeur de "TOP" de 1.
 5. Avant de push un élément, on vérifie que la Stack n'est pas pleine.
 6. Avant de pop un élément, on vérifie que la Stack n'est pas vide.
 Les bouttons en haut a gauche de votre navigateur permettant d'aller aux onglets précédents fonctionnent comme une stack.
 Votre navigateur va enregistrer tout vos onglets, lorsque vous cliquez sur le boutton retour, ce dernier vous affichera l'élément le plus en haut de la stack.

\ Files

Une file est une méthode d'algorithmie similaire aux queues que l'on peut retrouver dans les cinémas.
La personne tout devant aura son ticket avant la dernière personne.

La structure file respecte la règle FIFO (First In First Out), en effet, le premier élément est aussi le premier à sortir.

A savoir que la librairie Standard C++ propose une méthode de file.
La file possède une complexité de O(n). Il existe plusieurs méthodes associées a la file :
 -l'enfile => Ajout d'items dans la file
 -la défile => Enlèvement d'items de la file
 -Peek => Récuperer la valeur du devant de la file sans l'enlever (contrairement a la méthode "pop" qui elle l'enlève aussi).


Fonctionnement d'une file :
 - Deux pointeurs "FRONT" et "REAR" (front = devant ; rear = arrière)
 - Le pointeur "FRONT" traque le premier élément de la file
 - Le pointeur "REAR" traque le dernier élément de la file
 - Lors de l'init, on met la valeur de "FRONT" et "REAR" a -1


Opérations sur la file:
 - Enfile:
  - On vérifie si la file est pleine.
  - Pour le premier élément, on met la valeur du pointeur "FRONT" a 0.
  - On augmente l'indexe du pointeur "REAR" de 1.
  - Ajout de l'élément sur la position pointée par "REAR".

 - Défile :
  - On vérifie si la file est pleine.
  - On retourne la valeur pointée par "FRONT".
  - On augmente l'indexe du pointeur "FRONT" par 1.
  - Pour le dernier élément, on réinitialise les valeurs de "FRONT" et "REAR" a -1.
